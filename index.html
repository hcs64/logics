A brief explanation of

Modal logic

Very basic formal logics, the kind that are the most thoroughly studied, particularly by nonspecialists like me, deal with a single state of the universe. A proposition is true, or it is false. There is no sense of what it has been, or what it could be, or how it might get to be that way. A natural way to understand modal logic is that it allows us to work with alternate universes.

Consider the passage of time. We might want to say that p will be true at some time in the future. This would be expressed in temporal logic as Fp.

The F operator introduces another time relative to the current one, and all propositions in the context of that operator are making claims about a point in the future. There is also an operator G. Gp means p will always be true in the future, though note that that doesn't mean p is currently true. That is, it is not the case that $\text{F}p \rightarrow p$.

What is perhaps a bit more interesting is to say "$p$ will be true at some time in the future, and thereafter it will always be true". This would be expressed as $\text{F}(p \wedge \text{G}p)$.

You can see that F is interpreted as "at some time in the Future ...", while G is "it is always Going to be ...".
